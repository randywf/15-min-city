# 15 Minute City Presentation

## How to build

1. Install Quarto
3. Render the slides with: quarto render yourdirectory/presentation.qmd --to revealjs
4. Open the resulting html in a browser to present.

## Files

- `presentation.qmd` → Edit slide text and content here
- `styles2.css` → custom styling for slides
- `Backlog.tsv` → project backlog extracted from github, update as needed
